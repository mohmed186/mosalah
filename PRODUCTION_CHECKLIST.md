# Production Checklist

## الأمان
- تغيير جميع حسابات وكلمات مرور التجربة.
- تخزين الأسرار ومفاتيح الدفع في متغيرات بيئة.
- استخدام HTTPS وإعداد CSP وrate limiting وCSRF strategy.
- إضافة تحقق البريد والهاتف واستعادة كلمة المرور و2FA للمالك.
- مراجعة صلاحيات جميع API وإجراء penetration test.

## الأموال والدفع
- إنشاء حساب تاجر رسمي لدى بوابة الدفع.
- تطبيق payment intents وwebhooks والتحقق من التوقيع.
- عدم اعتبار الطلب مدفوعًا بناءً على استجابة المتصفح فقط.
- إضافة ledger مالي غير قابل للتعديل لتسويات البائعين.
- تحديد سياسة العمولة والاسترجاع والخصومات والضرائب.

## التشغيل
- PostgreSQL/MySQL مدارة مع نسخ احتياطية واختبار الاستعادة.
- تخزين الصور خارجيًا مع ضغط وتحجيم وفحص نوع الملف.
- logging مركزي ومراقبة uptime وأخطاء.
- بيئات Development / Staging / Production منفصلة.
- CI/CD واختبارات API وتكامل ودفع.

## الجوانب القانونية
- شروط استخدام للبائع والعميل.
- سياسة خصوصية وملفات تعريف الارتباط.
- سياسة شحن واسترجاع وحل النزاعات.
- التحقق من هوية البائع والبيانات الضريبية حسب القانون المحلي.
