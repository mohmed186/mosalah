# نشر Nest Marketplace على الإنترنت

## النتيجة
بعد النشر ستحصل أولًا على رابط عام شبيه بـ:

`https://nest-marketplace.onrender.com`

وبعد شراء دومين يمكن ربط رابط احترافي مثل:

`https://www.yourmarket.com`

## قبل البدء
أنشئ حسابين باسمك:

1. GitHub لحفظ كود الموقع.
2. Render لتشغيل الموقع وقاعدة البيانات.

## 1. رفع المشروع إلى GitHub

1. أنشئ Repository جديدًا Private.
2. ارفع **محتويات هذا المجلد** إلى المستودع، وليس الملف المضغوط نفسه.
3. تأكد أن `render.yaml` و`server.js` في جذر المستودع.

## 2. النشر من Render

1. افتح Render واختر **New > Blueprint**.
2. اربط حساب GitHub واختر مستودع المشروع.
3. سيقرأ Render ملف `render.yaml` تلقائيًا.
4. عند طلب المتغيرات السرية أدخل:
   - `OWNER_NAME`: اسمك أو اسم الإدارة.
   - `OWNER_EMAIL`: بريدك الذي ستدخل به إلى لوحة المالك.
   - `OWNER_PASSWORD`: كلمة مرور قوية لا تقل عن 12 حرفًا.
5. ابدأ النشر.
6. بعد نجاح النشر افتح رابط `onrender.com` الظاهر في لوحة Render.

> قاعدة البيانات محفوظة على Persistent Disk حتى لا تضيع الطلبات عند إعادة التشغيل أو تحديث الموقع.

## 3. الدخول إلى لوحة المالك

افتح:

`https://YOUR-SITE/owner.html`

وسجل بالبريد وكلمة المرور اللذين أدخلتهما أثناء النشر.

## 4. ربط دومين احترافي

1. اشترِ الدومين من شركة تسجيل دومينات باسمك.
2. داخل Render افتح الخدمة ثم **Settings > Custom Domains**.
3. أضف الدومين.
4. انسخ سجلات DNS التي يعرضها Render إلى لوحة شركة الدومين.
5. بعد اكتمال الربط يعمل HTTPS تلقائيًا.

## 5. إظهار الموقع في Google

1. أضف الدومين إلى Google Search Console.
2. أثبت الملكية عبر سجل DNS.
3. أرسل رابط خريطة الموقع:

`https://YOUR-DOMAIN/sitemap.xml`

4. استخدم URL Inspection لطلب فهرسة الصفحة الرئيسية.

ظهور الموقع بالرابط يكون فور نشره، أما ظهوره في نتائج البحث فليس فوريًا ولا تضمن Google ترتيبًا محددًا.

## ملاحظات مهمة قبل استقبال طلبات حقيقية

- غيّر اسم المنصة والعملة والعمولة من لوحة المالك.
- أضف شروط الاستخدام وسياسة الخصوصية والاسترجاع والشحن.
- اربط بوابة دفع بحساب تاجر فعلي.
- نفّذ نسخًا احتياطية دورية لقاعدة البيانات.
- عند نمو المنصة، انقل قاعدة البيانات من SQLite إلى PostgreSQL.
